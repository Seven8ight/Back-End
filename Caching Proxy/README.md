# Caching Proxy

Created project from roadmap.sh backend projects

## Setup

1. Install all necessary modules required outside the folder on the root folder

```bash
    npm i -g
```

2. Link the CLI application

```bash
    npm link
```

3. To run it on the terminal ensure to type in,

```bash
    Caching-proxy --port=<port> --origin=<url here>
```
