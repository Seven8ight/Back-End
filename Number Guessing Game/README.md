# Number guessing game

Created project from roadmap.sh backend projects

## Setup

1. Install all necessary modules required

```bash
    npm i -g
```

2. Build typescript files

```bash
    npm run build
```

3. Link the CLI Application(may require elevation of user priviledges on linux/mac platform)

```bash
    npm link
```

4. To run it on the terminal ensure to type in,

```bash
    Guessing-game
```

5. Execute
