# To do List API

Backend-oriented project from roadmap.sh

## Setup

### Server

1. Create a .env file and include the following variables

```bash
    PORT=<port number>
    POSTGRES_PASS=<password>
    JWT_ACCESS_TOKEN=<token here>
    JWT_REFRESH_TOKEN=<token here>
```

2. Run server file

```bash
    npx tsx --watch ./Todo\ List/\ API/Server/Server.ts
```

### Client

1. Install all necessary modules

```bash
    npm i -g
```

2. Execute

```bash
    npm run dev
```
